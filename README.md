# HelpDesk Operacional

Sistema de gestão de chamados e manutenção de facilities para grandes empresas.

> Projeto da disciplina **GCC267 - Projeto Integrador I** (UFLA, 2026/1).
> Documentação completa será adicionada durante o Sprint 1.

## Stack (visão inicial)

- **Backend**: Node.js + Express + TypeScript + Prisma
- **Frontend**: Next.js (App Router) + React + Tailwind CSS
- **Banco**: PostgreSQL
- **Filas/Cache**: Redis + BullMQ
- **Storage**: MinIO (S3-compatível)
- **Infra**: Docker + Docker Compose
